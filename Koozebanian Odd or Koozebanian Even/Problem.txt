The Koozebanian Phoob has just received homework about even and odd numbers but wants to write a program to do it for him.

Input Format

The first line of input contains the number of test cases that follow. Each test case contains a single number on a line by itself.

Constraints

There will be less than 1,000 test cases.

Output Format

For each test case, print out odd or even based on whether or not the answer is odd or even.

Sample Input 0:

3
1
6
212

Sample Output 0:

odd
even
even
